# Zentone

## Overview

Share Audio Enhanced Presentations

## Scope
#### Front End:
  - Handlebars.js
  - Bootstrap
  - CSS Grid
  - SCSS Precompiler
  - jQuery
#### Back End:
  - Node.js
  - Express.js
  - Sequelize ORM
  - PostgreSQL
  - Webpack
#### Libraries:
  - PDF.js
#### Hosting:
  - Aliyun OSS
  - Heroku
#### OAuth 2.0:
  - LinkedIn
  - Facebook
  - Twitter
